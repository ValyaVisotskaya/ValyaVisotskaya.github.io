ОНПУ. Украинско-немецкий институт

Украинский-немецкий учебно-научный институт осуществляет подготовку и переподготовку специалистов с высшим образованием с учетом опыта ведущих университетов Германии

О НАС

Преподавание на украинско-немецком факультете проводится лучшими преподавателями Одесского национального политехнического университета, владеющих немецким языком, ведущими профессорами немецких университетов.

На первых трех курсах студенты факультета одновременно с обучением по учебным планам соответствующих направлений интенсивно изучают немецкий язык. После третьего курса для студентов факультета организуется языковая практика в университетах Германии. С четвертого курса обучение студентов по специальным дисциплинам проводится на немецком языке. После четвертого курса для студентов факультета с помощью университетов Германии, заинтересованных предприятий Украины и немецких фирм организуется производственная практика на предприятиях немецких фирм. Защита дипломного проекта студентами факультета проводится на немецком языке.

Выпускники факультета получают украинский диплом бакалавра, специалиста, магистра установленного образца. Студенты, которые отлично успевают, могут быть рекомендованы для выполнения и защиты дипломного проекта в университетах Германии с получением немецкого диплома. Параллельно с получением диплома по основной специальности выпускники получают сертификат переводчика немецкого языка. Выпускники факультета рекомендуются для работы на совместных украинский-немецких предприятиях, на предприятиях Украины, использующих технику немецких фирм; украинских предприятиях, производящих продукцию для немецких и австрийских фирм; предприятиях иностранных фирм, которым необходимы специалисты, которые знают немецкий техническую и разговорный языки.
